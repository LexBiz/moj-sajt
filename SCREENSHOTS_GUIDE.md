# Как сделать скриншоты сайтов

## Быстрый способ - через браузер:

### Chrome/Edge:
1. Открой сайт (например https://sphynxdubai.ae/)
2. Нажми F12 (DevTools)
3. Ctrl+Shift+P
4. Напиши "screenshot"
5. Выбери "Capture full size screenshot"
6. Сохрани в `public/screenshots/`

### Или онлайн сервис:
Открой: https://www.screenshotmachine.com/
Введи URL и скачай скриншот
